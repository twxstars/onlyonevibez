## ðŸš€ Getting Started

### Prerequisites
- Node.js (v16 or higher)

### Installation
1.  **Clone the repository**
    ```bash
    git clone https://github.com/rahmon-tech/onlyonevibez.git
    cd onlyonevibez
    ```
2.  **Install dependencies**
    ```bash
    npm install
    ```
3.  **Start local server**
    ```bash
    npm run dev
    ```
    The site will launch at `http://localhost:5173`.
